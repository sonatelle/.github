# Sonatelle

Open-source works from memory into craft.

Sonatelle takes its name from *sonata*, a form where a theme leaves, returns, and is changed by the distance.

This space began from an unfinished feeling. Some things do not become the future once imagined for them; they remain as a phrase the heart still recognizes, returning in quieter keys. Here, that return is made into work: memory given structure, regret given patience, and care given something useful to build.

The projects here are small open-source works: careful tools, quiet experiments, and software shaped without spectacle.
